# 项目简介
本项目是基于使用Langchain调用大模型实现的智能诗歌学习网站拾诗，可以进行ai写诗，智能查诗，根据诗歌内容AI画图功能。本项目使用vue3搭建前端页面，fastapi连接前后端，实现前后端分离式运行。


# 使用说明
除了vue import部分，代码中所有路径均为绝对路径，请先换成自己的路径再运行

pycharm运行test目录下main.py文件

vscode在终端vue-01目录下运行
```
npm run serve
```
ctrl+鼠标左键点击生成链接
